Setup Guide
Database integration steps
Step 1 — Install dependency
npm install better-sqlite3

Ye SQLite ke liye fast synchronous driver hai. Koi alag DB server nahi chahiye.

Step 2 — Files place karo
📁 swiftshare/
├── server.js ← updated (DB integrated)
├── db.js ← naya file (DB layer)
├── package.json
├── .gitignore
└── public/
├── index.html
├── admin.html ← ye file (yahan ho)
└── swiftshare-engine.js
Step 3 — Run karo
npm start
# ya dev mode mein:
npm run dev

swiftshare.db automatically create ho jayega first run pe. Koi migration nahi chahiye.

Step 4 — GitHub pe upload
git init
git add .
git commit -m "feat: add SQLite DB + admin dashboard"
git remote add origin https://github.com/YOUR_USERNAME/swiftshare.git
git push -u origin main
Environment variables (optional)
PORT3000 (default)
DB_PATH./swiftshare.db (default)

Render pe deploy karte time: Settings → Environment Variables mein set karo.
